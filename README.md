## ✨ Fonctionnalités

- 🌓 **Mode sombre / clair** — Changement de thème avec `next-themes`
- 📱 **Responsive** — Optimisé mobile, tablette et desktop
- 💫 **Animations fluides** — Transitions avec `framer-motion`
- 📬 **Formulaire de contact** — Envoi sans backend via Web3Forms
- 🖼️ **Open Graph dynamique** — Aperçu personnalisé pour les réseaux sociaux

## 🛠️ Stack

| Catégorie | Technologie |

## 🚀 Démarrage local

```bash
git clone https://github.com/bilel-ghoudelbourk/portfolio.git
cd portfolio
pnpm install
```

Copie le fichier d'environnement et renseigne ta clé Web3Forms :

```bash
cp .env.example .env.local
```

Lance le serveur de développement :

```bash
pnpm dev
```

Ouvre [http://localhost:3000](http://localhost:3000) dans ton navigateur.
